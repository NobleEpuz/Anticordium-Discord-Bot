# LuniAssist Discord-bot
Multifunctional Discord bot for Relenova Roleplay.

## Features:
- AI-Tools (DALL-E 2/3, ChatGPT).
- System of wikipedia (reading .txt files with text with support of 2000+ symbols)
- Mail-system
- Economy system
- RP-questionnaire system
